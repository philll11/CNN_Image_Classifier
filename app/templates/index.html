Bear Classifier

This simple web app will take an image of a grizzly bear, black bear, and a teddy bear and classify which family of bear it belongs to.

It uses a Convolutional Neural Network with a Resnet34 layer architechure.
It was only trained on 600 images as it leverages the ImageNet library, allowing for fast, low volume training.
Tho it is a Neural Network, it does not require a GPU to run and therefore runs using the pytorch CPU distribution.
